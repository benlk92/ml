# AWS

After you ssh into the AWS server, run

````bash
   sudo apt-get install git
   git clone git@github.com:<username>/ml.git
   cd ml
   ./bootstrap.sh
````

# Local (Mac OSX)
Ensure you have Python 2.7.2 installed. Then run:
````bash
   git clone git@github.com:<username>/ml.git
   cd ml
   ./bootstrap.sh
